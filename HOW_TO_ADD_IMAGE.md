# Cara Menambahkan Gambar Kebaya

## Langkah-langkah:

1. **Buka folder project Anda** di File Explorer:
   ```
   E:\tugas prd tim b terbaru\public\images\
   ```

2. **Copy gambar kebaya Anda** ke folder tersebut

3. **Rename file menjadi:** `kebaya-product.jpg`
   - Jika file Anda adalah .png, rename menjadi `kebaya-product.png`
   - Jika file Anda adalah .jpeg, rename menjadi `kebaya-product.jpeg`

4. **Restart development server:**
   - Tekan `Ctrl+C` di terminal untuk stop server
   - Jalankan lagi: `npm run dev`

5. **Refresh browser** (tekan `F5` atau `Ctrl+R`)

## Format file yang didukung:
- `.jpg` atau `.jpeg`
- `.png`
- `.webp`

## Troubleshooting:

Jika gambar masih tidak muncul:
- Pastikan file ada di folder `public/images/`
- Pastikan nama file tepat: `kebaya-product.jpg` (case-sensitive)
- Coba hard refresh browser: `Ctrl+Shift+R` atau `Ctrl+F5`
- Restart development server
- Cek console browser (F12) untuk error messages
